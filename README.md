Monthly Budget Analyzer:

A command-line budgeting tool that helps users track expenses, monitor monthly spending, visualize category breakdowns, set budgets, and savings goals. Built with Python and designed for simplicity, accuracy, and personal finance awareness.

Features:

Add, view, and delete expenses across multiple categories

Spending summary with totals per category

Monthly summary table auto-grouped by year-month

Budget tracking with overspending alerts

Savings goal tracking

Progress checker comparing budget vs. actual spending

Color-coded charts (Red only for categories exceeding budget)

Persistent data storage via JSON

Tech Stack

Python 3

Matplotlib (Charts)

Tabulate (Formatted tables)

JSON (Data persistence)

How to Run:
python expense_tracker.py

Project Highlights:

Implemented a full CLI system with menus, input validation, and persistent storage.

Created dynamic visualizations for spending distribution using Matplotlib.

Designed a budget vs. spending logic system, including overspending detection.

Built a modular code structure for clean, maintainable functions.

Added a monthly grouping system using date parsing and accumulators.
